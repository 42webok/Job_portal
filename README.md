# 🧑‍💼 Job Portal - Laravel Project

A complete Job Portal Web Application built with Laravel.  
This system allows Employers to post jobs and Candidates to apply online.

---

## 🚧 Project Status

This project is currently under development.
New features and improvements are being added regularly.


## 🚀 Features

### 👤 Authentication
- User Registration & Login
- Employer Registration & Login
- Password Reset
- Profile Management

### 🏢 Employer Panel
- Post New Jobs
- Edit / Delete Jobs
- View Applicants
- Manage Company Profile

### 👨‍💻 Candidate Panel
- Browse Jobs
- Search & Filter Jobs
- Apply for Jobs
- Upload Resume
- Edit Profile

### 🛠 Admin Panel
- Manage Users
- Manage Employers
- Manage Jobs
- Approve / Reject Job Posts
- Dashboard Statistics

---

## 🛠 Built With

- Laravel
- PHP
- MySQL
- Bootstrap
- JavaScript / jQuery
- CropperJS (Profile Image Upload)

---


## 🔐 User Roles

| Role       | Description |
|------------|------------|
| Admin      | Full system control |
| Employer   | Post & manage jobs |
| Candidate  | Apply for jobs |
